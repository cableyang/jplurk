Provides classes and method for handling UID, Cookie, and Messages.

For using this library, we need HttpClient, Commons-logging and Commons-Codec libraries.

  • JAKARTA Commons HTTP Client
  • Apache Commons Logging
  • Apache Commons Codec
For JavaSE 5.0 API documentation, please see:
  • JavaTM 2 Platform Standard Edition 5.0 API Specification

@since JDK1.5
